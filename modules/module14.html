Module 14: Hosting & Git

Learn how to publish Mr.Elimu online using Git and hosting platforms like GitHub Pages.

Lesson Overview

This module introduces Git for version control, GitHub for collaboration, and GitHub Pages for hosting — so Mr.Elimu becomes a live, sovereign platform.

Hosting Steps

  1. Initialize Git in your project folder
  2. Create a GitHub repository
  3. Push your code to GitHub
  4. Enable GitHub Pages from repository settings
  5. Share your live link with the world

Reflection

Hosting is legacy. It turns your local vision into a global impact. Every learner deserves access.
