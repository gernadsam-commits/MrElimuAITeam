Module 3: Typography & Visual Hierarchy

Learn how to guide attention, create rhythm, and build emotional clarity through text.

Lesson Overview

This module explores font choices, heading sizes, spacing, and contrast — all tools for clarity.

Key Concepts

  • Font families — personality and readability
  • Hierarchy — using size and weight to guide the eye
  • Spacing — breathing room for the mind
  • Contrast — clarity through difference

Reflection

Typography is not decoration — it’s direction. Every learner deserves clarity, rhythm, and emotional peace.
